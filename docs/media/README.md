# README media assets

The root `readme.md` references the files below. Drop them here before publishing.
Two built-in Windows tools cover everything — no installs needed:

- **`Win + Shift + S`** — screenshots
- **`Win + G`** (Xbox Game Bar, `Win + Alt + R` to start/stop) — screen recording (mp4)

**Prep once:** set the UI to English (Settings → Language), use the dark theme, reset the playground, and run 2 clean flows so the lists have good-looking data.

## Required files

| File | What it shows | How |
| --- | --- | --- |
| `ghostly-demo.gif` | **The hero.** One new assisted run: goal → steps stream with green checks → **Success** verdict. Keep it snappy (~10–12s). | Record the run with `Win + Alt + R`. If the run is slow, **speed it up** when converting: at [ezgif.com](https://ezgif.com) → Video to GIF, set speed to 2–3x and/or trim, ~15 fps, width ~760, keep it 2–8 MB. |
| `home.png` | Home / overview page. | `Win + Shift + S` |
| `runs.png` | Runs list showing the three states (Success / Failure / Ghostly issue). | `Win + Shift + S` |
| `run-detail.png` | A run's detail: the unified step thread + snapshot + the verdict panel with its natural-language summary. | `Win + Shift + S` |

**Tip for a short GIF:** finish typing the goal first, then start recording (`Win + Alt + R`) right before you hit **Run** — so the clip is just run → verdict, not the typing.
