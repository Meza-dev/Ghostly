# README media assets

The root `readme.md` references the files below. Drop them here before publishing.
Two built-in Windows tools cover everything — no installs needed:

- **`Win + Shift + S`** — screenshots
- **`Win + G`** (Xbox Game Bar) — screen recording (mp4)

**Prep once:** reset the playground and run 2 clean flows so the dashboard has good-looking data. Use the dark theme.

## Required files

| File | What it shows | How |
| --- | --- | --- |
| `ghostly-demo.gif` | **The hero.** ~12s loop of one assisted run: open **New run** → goal → steps stream with green checks → a selector self-heals → **Success** verdict. | Record the run with `Win + G`, then trim ~12s and convert to GIF at [ezgif.com](https://ezgif.com) (keep it 2–8 MB, ~15 fps). |
| `dashboard.png` | Runs list showing the three states (Success / Failure / Ghostly issue). | `Win + Shift + S` |
| `run-detail.png` | A run's detail: the unified step thread + snapshot + the verdict panel with its natural-language summary. | `Win + Shift + S` |

## Demo video (hosted, not committed)

Record one ~60s clip with `Win + G`: connect the AI in **Settings → Assisted mode**, create an assisted run, watch it self-heal, and land the verdict. Upload it to **YouTube (unlisted)** or Vimeo, then replace `REPLACE_WITH_VIDEO_ID` in `readme.md` with the video URL. Grab a frame from the clip if you want a custom thumbnail.
